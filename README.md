# Trading Bot - Polymarket Prediction Markets

## ⚠️ Safety First
This bot operates under STRICT safety rules:
- NO auto-trading until Phase 3
- Alert + Manual Approval required for all trades
- Max $1 per trade initially
- Kill switch always active

## Phases
1. **Phase 1:** Backtest & Paper Trading (current)
2. **Phase 2:** Extended Paper (2+ weeks)
3. **Phase 3:** Live Trading (requires explicit approval)

## Quick Start
```bash
cd src
python3 bot.py --help
```
